# Drinks Catalog

## Configuração


### Instale as dependencias:
- Node ^16.10
ou
- Node ^14.18

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

### Inicializando servidor

Será inicializado em `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```

## Build


```bash
# npm
npm run build

# pnpm
pnpm run build

# yarn
yarn build
```

##### Preview local do build:

```bash
# npm
npm run preview

# pnpm
pnpm run preview

# yarn
yarn preview
```
